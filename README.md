# Veyra — ERP Greder Montagen

ERP interne pour la gestion des chantiers, du pointage et de la facturation.

**Stack :** Angular · NestJS · PostgreSQL · Prisma · Docker · pnpm workspaces

## Structure

```
apps/web      Frontend Angular
apps/api      Backend NestJS
packages/     Contrats partagés
docs/         Documentation et ADR
infra/        Infrastructure Docker
```

## Prérequis

- Node >= 20
- pnpm >= 9
- Docker Desktop

## Démarrage

> À compléter lors de l'initialisation des applications (Phase 1).
